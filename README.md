# n8n Automation Workflows

A collection of workflow automation projects built using **n8n**, **JavaScript**, **Gemini AI**, **Gmail API**, **People Data Labs API**, **JSONBin**, and REST APIs.

This repository showcases practical workflow automation concepts including AI-powered email generation, API integration, lead enrichment, data transformation, and intelligent workflow decision-making.

# 🚀 Projects

## 📧 AI-Email-Automation

Generate personalized emails using **Gemini AI** and automatically deliver them through **Gmail**.

### Features

- Form Submission
- Gemini AI Integration
- AI Generated Email Content
- Gmail Automation
- OAuth Authentication

**Workflow**

```
Form Submission
        ↓
Gemini AI
        ↓
Generate Email
        ↓
Send via Gmail
```

      ↓

      ↓

Generate Qualified Leads

## 🤖 Intelligent-Lead-Qualification

An intelligent workflow that determines whether an email belongs to a company domain and automatically sends either a personalized proposal or a welcome email.

### Features

- Form Submission
- JavaScript Logic
- Company Domain Detection
- Conditional Workflow
- People Data Labs Integration
- Personalized Proposal Emails
- Generic Welcome Emails

**Workflow**

```
User Form
      ↓
Extract Email Domain
      ↓
Company Domain?
   ↙            ↘
Yes              No
 ↓                ↓
People Data API   Welcome Email
 ↓
Personalized Proposal
```
